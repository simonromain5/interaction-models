Active brownian motion¶

class activebrownianparticles.Vicsek(v, n_particles, noise, dt=20, radius=1, surface=10000, n_steps=2000, janus=False, stop=False)¶

This class represents the motion of particles according to the Vicsek model in 2 dimensions. The motion of the particles is restricted in a box and is not elastic. If stop is set to True, then the particles stop at each contact. In this case, they separate in the same conditions as in the BallStop class.

Parameters:
  • v (float or int) – Speed of the particle
  • n_particles (int) – Number of particles in the box
  • dt (float, optional) – 20 by default. Increment of time for each step.
  • radius (float, optional) – 1 by default. radius of the particles. It as constant for all the particles
  • surface (float, optional) – 10000 by default. Surface of the box. Box is a square, hence length_side = square_root(surface)
  • n_steps (int, optional) – 2000 by default. Number of steps that we consider for the total movement of the particles.
  • janus (bool, optional) – False by default. Particles are janus particles if set to True.
  • stop (bool, optional) – stop the particle each time it encounters another one.

update_velocities(contact_pairs=None, contact_index=None)¶

This function updates the velocities of each particle considering the definition of the Vicsek model. Each particle changes its angle to align itself with is neighbors, hence the velocities vectors are updated at each step. :param contact_pairs: Default is None (if self.stop is set to False). Array of all the pairs of contact [i, j]. Shape is (n_contacts, 2). :type contact_pairs: np.array, optional :param contact_index: Default is None (if self.stop is set to False). Index of the particles in contact. :type contact_index: np.array
